# ✨ Shimmer Loading Skeleton (SwiftUI)

A reusable SwiftUI shimmer loading system for building skeleton screens with smooth animated placeholders.

## ✨ Features

* Skeleton loading UI with shimmer effect
* Fully customizable shimmer (direction, gradient, speed)
* Supports light & dark mode
* Custom corner radius & specific corner masking
* Easy-to-use `loadingStyle` modifier
* Works with any SwiftUI view

## 🧩 Architecture

* `Shimmer` → core shimmer animation
* `LoadingModifier` → combines shimmer + redaction
* `FakeLoadingView` → demo skeleton screen

## 🚀 Tech

SwiftUI • ViewModifier • Animations • Gradient • Redacted API

A flexible and clean solution for modern loading states.
